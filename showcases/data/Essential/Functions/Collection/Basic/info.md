---
title: Functions - Collection (Basic)
description: Common collection functions - filter, map, exists, forAll, fold, distinct, sortBy, size.
---

These functions operate on any Pure collection (`T[*]`). They form the workhorses of
non-relational logic: lambda-driven element-wise transformations (`map`, `filter`),
reductions (`fold`, `sum`, `count`), boolean tests (`exists`, `forAll`) and
deduplication / ordering (`distinct`, `sortBy`).
