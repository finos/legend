---
title: H2 Connection (LocalH2)
description: Showcase of a LocalH2 RelationalDatabaseConnection seeded with inline test data, ideal for unit tests and quick demos.
---

`LocalH2` runs an in-memory H2 database that can be pre-seeded via `testDataSetupSqls`.
This is the connection type used by mapping tests and function tests across the
showcases because it requires no external infrastructure.
